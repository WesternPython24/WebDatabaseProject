This website will help game masters and creative storytellers create a world and invite others to this world. Different people can be invited and they will have set permissions to view/edit/add files. The author of a story will be able to make connections between different ideas/characters/locations. 


This is the Database design. It keeps track of everything from users, stories, characters, notes, permissions, etc
